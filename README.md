# Task 04 – Business Insights Report (EDA)

## 📌 Objective
Perform Exploratory Data Analysis (EDA) on a marketing campaign dataset.  
Instead of building a Machine Learning model, the focus is on **why** certain channels perform better.  
The goal is to clean the data, visualize the funnel, and recommend budget allocation based on ROI.

## 📊 Deliverables
- **Power BI Dashboard (.pbix)**  
  - Funnel visualization (Impressions → Clicks → Leads → Conversions → Revenue)  
  - ROI by channel  
  - Yearly conversions and revenue trends  
- **Word Report (.docx / PDF)**  
  - 1-page summary of insights and recommendations  

## 🔎 Key Insights
- **Email Campaigns** → ROI = **5.0** → Best performer  
- **Google Ads & LinkedIn Ads** → ROI = **1.5** → Moderate efficiency  
- **Facebook & Instagram Ads** → ROI = **0.1** → Poor efficiency  
- Funnel analysis shows natural drop-offs, but conversions → revenue transition is strong  
- 2024 totals (~70K) highlight healthy monetization once leads convert  

## 🎯 Recommendations
- Increase budget for **Email Campaigns**  
- Maintain moderate spend on **Google Ads** and **LinkedIn Ads**  
- Reduce spend on **Facebook** and **Instagram Ads** until strategies are optimized  

## 📌 Conclusion
This project demonstrates how Exploratory Data Analysis (EDA) can uncover actionable insights from marketing campaign data.  
By combining a **Power BI dashboard** with a **summary report**, the analysis provides both visual clarity and strategic recommendations.  
The findings emphasize the importance of **data-driven decision-making** in marketing, ensuring resources are allocated to the most effective channels.
